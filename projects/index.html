Scheme!

Interpreter for Scheme (lisp dialect) written with Python. Features usage of buffer and stack data structures. Properly applies tail recursive function calls so that it remains correct with proper Scheme convention. Very tough project and took extensive critical thinking, teamwork, and patience (the debugging was brutal). Includes exhaustive test cases.
